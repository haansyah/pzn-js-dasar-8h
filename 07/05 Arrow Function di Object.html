Arrow function di object

  • Sebelumnya kita sudah membahas tentang arrow function
  • Arrow function juga bisa kita gunakan sebagai object method
  • Namun perlu diketahui, Arrow function tidak bisa digunakan untuk mengakses arguments object, function generator, kata kunci this, dan kata kunci super
  • Jadi pastikan menggunakan arrow function hanya memang ketika kita tidak butuh fitur-fitur diatas
