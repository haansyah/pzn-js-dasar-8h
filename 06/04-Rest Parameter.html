Rest Parameter

  1. Rest Parameter adalah fitur dimana kita bisa mengirim data sebanyak-banyak nya pda satu parameter, dan secara otomatis akan dikonversi menjadi array
  2. Untuk membuat rest parameter, ada ketentuannya
  3. Rest parameter hanya boleh ada satu di function, tidak boleh lebih dari satu
  4. Rest parameter hanya boleh berada diposisi paling akhir, tidak boleh di depan/ tengah, kecuali memang cuma ada 1 param
  5. di bahasa pemrogramman lain, ada juga yang bilang ini adalah variable argument
